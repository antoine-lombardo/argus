# Architecture Decision Records

Use this folder when locking a design choice (plugin runtime, packaging, auth model, etc.).

Suggested filename: `NNNN-short-title.md` (e.g. `0001-plugin-runtime.md`).

Minimal template:

```markdown
# Title

Date: YYYY-MM-DD
Status: Proposed | Accepted | Superseded

## Context

## Decision

## Consequences
```

Link accepted ADRs from [../PLUGIN-SYSTEM.md](../PLUGIN-SYSTEM.md) or [../DEVELOPMENT.md](../DEVELOPMENT.md) when relevant.
